## Link API

### Link Attributres

| 属性名    | 描述           | 类型                                               | 默认值  |
| --------- | -------------- | -------------------------------------------------- | ------- |
| type      | 类型           | `default/ primary/ success/ info/ danger/ warning` | default |
| disabled  | 是否为禁用状态 | `boolean`                                          | false   |
| underline | 是否具有下划线 | `boolean`                                          | true    |
| href      | 跳转的链接     | `string`                                           | -       |
